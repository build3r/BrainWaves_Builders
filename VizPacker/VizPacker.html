Deploy to SAP Lumira

  1. Unzip the .zip file; for example, <extension_name>.zip
  2. Modify extension.js based on needs; for example, add implementation code to draw shapes in the plot area. (1)
  3. Copy the whole unzipped folder into SAP Lumira extensions folder; for example, copy the folder of <extension_name> into “ C:\Program Files\SAP Lumira\Desktop\extensions”;
  4. Open SAP Lumira; the extension can be chosen from the chart type list.
(1)Packer only generates the extension structure and startup scripts for creating an extension. You might need to modify extension.js according to your needs before deploying to SAP Lumira.

What is VizPacker ?

VizPacker is a handy tool that helps visualization developers easily design, build, and preview a chart based on CVOM SDK. VizPacker will auto-create a package for the CVOM chart extension; the package includes a set of fundamental codes and files based on the visualization module and data schema.

VizPacker is meant to help users to quickly use the implementation workflow and avoid unnecessary issues or problems.

What's next ?

After getting the packed extension zip bundle, you are free to unzip it and switch to another code editor, such as Notepad or Eclipse, for further development. Once you've finished, you are able to deploy the extension to Lumira and debug it in Lumira debug mode.

If you want to debug the extension in web browsers locally first before deploying it into Lumira,all of the required libraries are under directory
“<SAP Lumira installation directory>/Desktop/libs/”.
